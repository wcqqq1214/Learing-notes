- **date**    ( 显示当前日期 )
	基本语法:
	1. **date**             ( 显示当前时间 )
	2. **date +%Y**    ( 显示当前年份 )
	3. **date +%m**   ( 显示当前月份 )
	4. **date +%d**    ( 显示当前是哪一天 )
	5. **date "+%Y-%m-%d %H:%M%S"**    ( 显示年月日时分秒 )
	6. **date -s 字符串时间**    ( 设置系统当前时间 )
	案例:
		设置系统当前时间，如设置成 2024-11-12 17:16:56
		**date -s "2024-11-12 17:16:56"**	
***ps:    注意大小写 ! ! !***

- **cal**    ( 查看日历 )    ( calendar )
	基本语法:
		**cal \[ 选项 ]**        ( 不加选项，显示本月日历 )
	案例:
		**cal 2024**        ( 显示2024年的日历 )
