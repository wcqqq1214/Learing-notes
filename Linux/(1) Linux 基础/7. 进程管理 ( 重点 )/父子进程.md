***ps:***    
1. *每个进程都有一个父进程，该父进程创建并管理这个进程*
2. *如果停止了父进程，子进程也会随之停止*


- **ps -ef** 是以全格式显示当前所有的进程
	( **-e** 显示所有进程 `everyone`，**-f** 全格式 `full` )
	1. **UID**        用户 ID
	2. **PID**        进程 ID
	
	3. **PPID**      **父进程 ID**    `parent process id`
	4. **C**            [[CPU 使用优先级因子]]
		1. **数值越大**，表明进程是 **CPU 密集型运算**，执行**优先级降低**
		2. **数值越小**，表明进程是 **I/O 密集型运算**，执行**优先级提高**
		
	5. **STIME**    进程的启动时间
	6. **TTY**        完整的终端名称
	7. **TIME**      使用 CPU 的时间
	8. **CMD**      启动进程所用的命令和参数
