`备份和恢复基本介绍`

实体机无法做快照，如果系统出现异常或者数据损坏，后果严重，需要重装系统，还会造成数据丢失，所以要使用备份和恢复技术

- Linux 的备份和恢复有两种方式 :
	1. 把需要备份的文件 ( 或者分区 ) 用 `tar` 打包，下次需要恢复的时候，再解压开覆盖即可
	2. 使用 `dump` 和 `restore` 命令

- 安装 **dump** 和 **restore**
	1. **yum -y install dump**
	2. **yum -y install restore**
	( `-y` 用于 **自动回答 "是"** ( yes ) 给所有提示的问题 )
